Jade 隐私政策

最近更新日期：2026年3月20日

感谢您选择 Jade 浏览器扩展（以下简称“本插件”）。我们非常重视您的隐私。本政策旨在透明地说明我们如何处理您的数据。

1. 核心原则

Jade 遵循 “本地优先” 的设计理念。我们的核心准则如下：

  • 不收集个人信息：我们不收集您的姓名、电子邮件地址、电话号码或任何可识别身份的信息。
  • 数据本地存储：您的书签、浏览历史以及插件配置均存储在您的本地浏览器中，绝不上传至任何第三方服务器。
  • 无跟踪：我们不包含任何追踪脚本、广告插件或分析像素。

2. 权限说明与数据使用

为了提供核心功能，Jade 请求了以下权限：

  • 书签 (Bookmarks)：用于展示、整理和维护您的书签。
  • 历史记录与会话 (History & Sessions)：仅用于在“最近访问”模块中展示您最近浏览过的页面，方便快速找回。
  • 本地存储 (Storage)：用于保存您的个人化设置（如壁纸偏好、隐藏的URL列表）。
  • 主页权限 (New Tab)：用于将新标签页替换为简洁的沉浸式壁纸页面。

3. 外部服务调用

本插件仅在以下特定场景下发起外部网络请求：

  • 获取壁纸：请求 Bing、NASA 以及 Picsum 等平台获取当日精选或随机壁纸。
  • 获取图标 (Favicon)：通过 Chrome 内部 API 或 Google Favicon 服务获取网站的小图标。
  • 翻译功能：若您使用插件内的翻译工具，相关文本将发送至翻译服务供应商提供的 API（如 Google Translate）。
  • NASA API Key：插件默认使用公开的 DEMO_KEY。若您主动填写自己的 NASA API Key，该配置仅保存在本地浏览器存储中，不会上传给项目作者。

4. 数据共享与披露

由于我们不收集您的数据，因此不存在将数据出售或共享给第三方营利的情况。我们绝不会因商业目的披露您的本地浏览偏好。

5. 您的权利

由于所有数据均存储在您的本地浏览器中，您可以随时通过以下方式管理数据：

  • 清除浏览器缓存和插件数据。
  • 在设置界面导出或重置配置。
  • 从浏览器中卸载插件，届时所有插件相关的本地配置将被自动删除。

6. 联系我们

如果您对本隐私政策有任何疑问，请通过 GitHub 仓库提交 Issue 或联系项目维护者。
