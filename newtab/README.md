# Pure New Tab | 极简沉浸新标签页

这是一个极具设计感、追求极致简约与沉浸式体验的 Chrome 新标签页扩展。它消除了所有冗余的视觉干扰，仅保留最核心的功能，让你在每次打开浏览器时都能拥抱一份纯净与灵感。

## ✨ 核心特性

### 1. 🖼️ 精选壁纸库
支持多种高质量图片来源，满足不同的审美需求：
- **必应主页 (Bing)**：同步全球每日必应精选，通过镜头看世界。
- **NASA 天文 (NASA)**：探索宇宙深处，每日更新壮丽的星云与星系图景。
- **无尽随机 (Picsum)**：海量高清摄影作品随机切换，每次都有新发现。
- **我的收藏 (Favorites)**：将你心动的每一瞬壁纸永久保存，并支持收藏图片轮播。

### 2. 🖱️ 极简交互逻辑
- **隐藏式菜单**：主界面仅保留【收藏】与【刷新】两个极微小按钮，甚至连设置按钮都已被整合。
- **灵动气泡选择器**：鼠标悬停在刷新按钮上 **1秒**，上方会优雅唤起透明毛玻璃材质的选择菜单。
- **防误触设计**：特设的悬停延迟机制，确保在快速操作或鼠标划过时不会打断你的思路。

### 3. 📖 沉浸式图说
- **全屏故事卡片**：点击左下角的版权信息，底部会升起沉浸式的背景故事。
- **宽屏优化排版**：故事文字采用宽屏适配逻辑，阅读体验如同翻开一本精美的电子杂志。

### 4. ⚡ 极致性能
- **原生技术栈**：基于 HTML5 / CSS3 / Vanilla JS 构建，不依赖任何重型框架。
- **极速响应**：毫秒级启动，完全不降速，甚至比 Chrome 默认标签页更轻快。

---

## 🚀 安装与使用

1. 在 Chrome 地址栏输入 `chrome://extensions/`。
2. 开启右上角的 **开发者模式**。
3. 点击 **加载已解压的扩展程序**，选择本项目所在的文件夹。
4. 打开新标签页，开启你的沉浸之旅。
